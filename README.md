# Shriniwas Portfolio

A personal developer portfolio built with React and Vite to showcase my skills, projects, education, and achievements.

## Tech Stack

- React
- Vite
- JavaScript
- CSS3

## Features

- Responsive portfolio layout
- Hero section with profile image
- Skills-focused tech stack section
- Project showcase
- Education and achievements sections
- Contact section
- Downloadable resume
- Theme toggle

## Getting Started

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

## Project Structure

```text
src/
  components/
  App.jsx
  index.css
  main.jsx
public/
```
